# Procedimiento de Ensamblaje - Permeámetro Mejorado

## Objetivo
Guía paso a paso para el ensamblaje completo del permeámetro con sistema de sellado mejorado.

---

## Pre-requisitos

### Herramientas Necesarias

**Básicas:**
- [ ] Llaves Allen (set métrico 2-8mm)
- [ ] Destornilladores (plano y Phillips)
- [ ] Llaves españolas (8, 10, 13mm)
- [ ] Torquímetro (rango 2-10 Nm)
- [ ] Cúter o navaja afilada
- [ ] Calibrador vernier (pie de rey)

**Recomendadas:**
- [ ] Pistola de calor (para insertos)
- [ ] Nivel de burbuja
- [ ] Paño de microfibra
- [ ] Bandeja de trabajo

**Materiales de Consumo:**
- [ ] Alcohol isopropílico (250ml)
- [ ] Lubricante de silicona
- [ ] Cinta PTFE (teflón)
- [ ] Guantes de nitrilo
- [ ] Toallas absorbentes

## FASE 1: Preparación de Componentes

### Paso 1.1: Limpieza Inicial

**Objetivo:** Eliminar residuos de fabricación, aceites y partículas.

```
PROCEDIMIENTO:

1. Preparar solución de limpieza:
   - Agua tibia + jabón pH neutro
   - Evitar detergentes agresivos

2. Lavar todos los componentes de plástico:
   - Sumergir en solución 5 minutos
   - Cepillar suavemente con cepillo suave
   - Enjuagar abundantemente con agua destilada

3. Limpiar componentes metálicos:
   - Desengrasar con alcohol isopropílico
   - Secar con paño limpio
   - Verificar ausencia de corrosión

4. Secado:
   - Aire comprimido filtrado (ideal)
   - O dejar secar al aire en ambiente limpio
   - NO usar trapos que suelten pelusa

5. Inspección visual:
   - Sin partículas adheridas
   - Sin aceites o grasas
   - Sin daños mecánicos
```

**IMPORTANTE:** No usar acetona ni solventes agresivos en acrílico o PVC.

### Paso 1.2: Inspección de Superficies de Sellado

**Tapa Superior:**

```
1. Verificar superficie de contacto:
   - Colocar regla metálica de precisión
   - Buscar luz entre regla y superficie
   - Planicidad < 0.05mm (ideal)

2. Inspeccionar canal de O-ring:
   - Sin rebabas ni asperezas
   - Profundidad uniforme
   - Esquinas redondeadas (R 0.5mm)

3. Verificar perforaciones de pernos:
   - Alineación correcta (PCD 95mm)
   - Sin grietas alrededor
   - Diámetro 6.5mm (+0.2mm tolerancia)
```

**Borde Superior del Cuerpo:**

```
1. Verificar perpendicularidad:
   - Usar escuadra de precisión
   - Máximo 0.1mm de desviación

2. Verificar planicidad:
   - Igual que con la tapa
   - Esta es la superficie MÁS CRÍTICA

3. Si no es satisfactorio:
   - Lijar manualmente sobre vidrio plano
   - Papel de agua P1000 → P2000
   - Verificar frecuentemente
```

### Paso 1.3: Preparación de Sellos

**O-ring Principal:**

```
1. Inspección visual:
   ✓ Sin cortes o rasgaduras
   ✓ Sin deformaciones permanentes
   ✓ Sección transversal uniforme
   ✓ Color uniforme (negro o marrón)
   ✗ Desechar si presenta defectos

2. Limpieza:
   - Lavar con agua y jabón neutro
   - Enjuagar completamente
   - Secar con paño sin pelusa

3. Lubricación:
   - Aplicar lubricante de silicona
   - Capa delgada y uniforme
   - Cubrir toda la superficie
   - Usar guantes limpios
```

**Empaque Plano PTFE:**

```
1. Verificar dimensiones:
   - Ø exterior: 70mm (±0.5mm)
   - Ø interior: 63.5mm (+0.5mm / -0mm)
   - Espesor: 2mm (±0.1mm)

2. Inspección:
   - Sin grietas o roturas
   - Bordes limpios y rectos
   - Superficie lisa y uniforme

3. Limpieza:
   - Limpiar con alcohol isopropílico
   - Secar completamente
   - NO lubricar (el PTFE es auto-lubricado)
```

## FASE 2: Instalación de Insertos y Puertos

### Paso 2.1: Instalación de Insertos Roscados

**Método 1: Instalación Térmica (RECOMENDADO)**

```
PARA TAPA SUPERIOR:

1. Preparación:
   - Marcar centro de perforación
   - Perforar piloto según inserto (típ. Ø10mm)
   - Limpiar virutas completamente

2. Calentamiento del inserto:
   - Usar pistola de calor a 180-200°C
   - Calentar inserto 20-30 segundos
   - Sostener con pinzas (¡está caliente!)

3. Instalación:
   - Alinear inserto con perforación
   - Presionar firmemente con herramienta plana
   - Mantener presión hasta que enfríe (1 min)
   - Verificar que brida quede asentada

4. Enfriamiento:
   - Dejar enfriar 5 minutos completamente
   - El plástico se contrae y agarra el inserto
   - NO manipular mientras esté caliente

5. Verificación:
   - Intentar girar inserto (no debe moverse)
   - Verificar perpendicularidad
   - Rosca debe estar limpia y completa
```

**Método 2: Instalación con Rosca (Alternativo)**

```
Para insertos con rosca externa:

1. Perforar diámetro específico del inserto
2. Aplicar adhesivo para plástico (opcional)
3. Atornillar inserto con destornillador o llave
4. Apretar hasta que brida asiente
5. Limpiar exceso de adhesivo
6. Dejar curar según fabricante
```

### Paso 2.2: Instalación del Difusor de Entrada

```
              TAPA
         ┌────────────┐
         │   Inserto  │
         └─────┬──────┘
               │
         ┌─────▼──────┐
         │  DIFUSOR   │ ← Instalar aquí
         │  ●●●●●●●   │
         └────────────┘

PROCEDIMIENTO:

1. Verificar que difusor tiene 4 agujeros de montaje

2. Alinear difusor concéntrico con inserto

3. Usar tornillos M3 × 10mm (acero inox):
   - Arandelas M3 bajo cabeza
   - Apretar en patrón diagonal
   - Torque: 0.5 Nm (suave)

4. Verificar que difusor queda paralelo a la tapa

5. Comprobar que perforaciones no están obstruidas
```

### Paso 2.3: Instalación de la Rejilla Filtrante

```
Ubicación: En la base o inicio del cuerpo

1. Si la base es separada:
   - Colocar rejilla sobre la salida
   - Puede pegarse con adhesivo de silicona
   - O usar reborde mecánico

2. Si la rejilla va dentro del cuerpo:
   - Insertar rejilla con soporte
   - Dejar 10mm desde el fondo (cámara de sedimentación)
   - Fijar con pequeños tornillos laterales

3. Verificación:
   - Rejilla debe estar horizontal (usar nivel)
   - Sin inclinación > 2°
   - No obstruye puerto de salida
```

## FASE 3: Ensamblaje del Sistema de Sellado

### Paso 3.1: Colocación del O-ring

**CRÍTICO:** Este paso determina el éxito del sellado.

```
PROCEDIMIENTO:

1. Limpiar canal del O-ring en la tapa:
   - Alcohol isopropílico
   - Secar con aire comprimido
   - Inspeccionar con linterna

2. Aplicar lubricante al O-ring:
   - Lubricante de silicona
   - Extender uniformemente con dedos (guantes)
   - Capa muy delgada

3. Colocar O-ring en el canal:
   - Comenzar en un punto
   - Ir colocando circularmente
   - NO estirar excesivamente
   - Evitar torsiones

4. Verificar asentamiento:
   - O-ring debe quedar completamente en el canal
   - Sin secciones levantadas
   - Distribución uniforme

5. Inspección final:
   - Pasar dedo alrededor (con guante)
   - Sentir que está bien asentado
   - Sin irregularidades
```

### Paso 3.2: Posicionamiento del Empaque Plano

```
1. Limpiar borde superior del cuerpo:
   - Eliminar polvo o partículas
   - Verificar que esté seco

2. Colocar empaque de PTFE:
   - Centrar sobre el borde del cuerpo
   - Diámetro interior debe coincidir
   - Debe quedar plano, sin arrugas

3. NO aplicar lubricante al PTFE

4. El empaque debe cubrir uniformemente el borde
```

## FASE 4: Ensamblaje Tapa-Cuerpo

### Paso 4.1: Alineación Inicial

```
1. Verificar marcas de alineación:
   - Tapa y cuerpo tienen marcas grabadas
   - Alinear las marcas

2. Posicionar tapa sobre cuerpo:
   - Bajar verticalmente (no inclinar)
   - Centrar visualmente
   - Puede haber ligera resistencia del O-ring

3. Asentar tapa:
   - Presionar suavemente con ambas manos
   - Distribuir presión uniformemente
   - NO golpear
```

### Paso 4.2: Inserción de Pernos

```
1. Preparar pernos:
   - Limpiar roscas con cepillo
   - Verificar que no están dañados

2. Insertar los 8 pernos con arandelas:
   - Comenzar por cualquier perno (ej. posición 1)
   - Insertar perno con arandela
   - Atornillar a mano 2-3 vueltas
   - NO apretar aún

3. Secuencia de inserción:
   - Insertar perno opuesto (posición 5)
   - Luego pernos 3 y 7
   - Finalmente 2, 4, 6, 8
   
        1
    8       2
   7    ●    3
    6       4
        5

4. Todos los pernos deben girar libremente
```

### Paso 4.3: Secuencia de Apriete

**CRÍTICO:** Seguir secuencia estrella para distribución uniforme.

```
PRIMERA PASADA - Apriete Suave:

1. Torque objetivo: 2 Nm

2. Secuencia en estrella:
   1 → 5 → 3 → 7 → 2 → 6 → 4 → 8

3. Usar torquímetro calibrado:
   - Ajustar a 2 Nm
   - Apretar hasta que "click"
   - NO exceder

4. Verificar planicidad:
   - Tapa debe quedar paralela al cuerpo
   - Sin inclinación visible

SEGUNDA PASADA - Apriete Medio:

1. Torque objetivo: 4 Nm

2. Misma secuencia estrella:
   1 → 5 → 3 → 7 → 2 → 6 → 4 → 8

3. El O-ring ahora está comprimido ~20%

TERCERA PASADA - Apriete Final:

1. Torque objetivo: 5 Nm (máximo)

2. Misma secuencia estrella

3. Verificación final:
   - Medir separación con calibrador (debe ser 0)
   - Todas las tuercas al mismo nivel
   - Tapa completamente paralela
```

**ADVERTENCIA:** NO exceder 5 Nm. Sobre-apretar puede:
- Romper el acrílico
- Extruir el O-ring
- Deformar la tapa

### Paso 4.4: Inspección Post-Ensamblaje

```
CHECKLIST:

□ Tapa paralela al cuerpo
□ Sin separación visible
□ Todos los pernos al mismo nivel
□ O-ring no está visible (comprimido)
□ No hay deformación del plástico
□ Marcas de alineación coinciden
```

## FASE 5: Instalación del Sistema de Plomería

### Paso 5.1: Instalación de Válvulas

**Válvula de Aguja (Entrada):**

```
1. Aplicar cinta PTFE:
   - 3-4 vueltas en sentido horario
   - Comenzar 2 hilos desde el final
   - Presionar mientras enrolla

2. Atornillar en inserto de tapa:
   - A mano primero
   - Dar 1.5 vueltas con llave
   - NO forzar (rosca en plástico)

3. Orientar manija hacia operador

4. Verificar operación:
   - Abrir y cerrar varias veces
   - Debe operar suavemente
```

**Válvula de Bola (Salida):**

```
1. Mismo procedimiento que válvula de aguja

2. Instalar en puerto de salida inferior

3. Orientar manija para fácil acceso

4. Verificar:
   - Posición abierta: 90° (paralela al flujo)
   - Posición cerrada: 0° (perpendicular)
```

**Válvula de Venteo:**

```
1. Instalar en puerto superior central

2. Orientar hacia arriba o lateral

3. Conectar tubo pequeño para descarga controlada

4. En posición cerrada antes de uso
```

### Paso 5.2: Instalación de Manómetros

```
1. Colocar conectores T en línea:
   - T1: Antes de entrada a columna
   - T2: Después de salida de columna

2. Instalar manómetros en derivación de T:
   - Aplicar cinta PTFE en rosca
   - Atornillar a mano
   - Dar 1 vuelta adicional con llave

3. Orientar displays:
   - Hacia el operador (frente)
   - Fácilmente legibles

4. Verificación:
   - Sin presión debe leer 0.00 bar
   - Si no: función ZERO o TARE
```

### Paso 5.3: Conexión de Tubería

```
TUBO DE ENTRADA (Bomba → Columna):

1. Medir longitud necesaria:
   - Desde bomba hasta válvula de aguja
   - Dejar 10% adicional para curvas

2. Cortar tubo:
   - Usar cúter afilado
   - Corte perpendicular (90°)
   - Limpiar rebabas

3. Conectar con push-fit:
   - Insertar hasta el tope (se siente)
   - Tirar para verificar (no debe salir)

TUBO DE SALIDA (Columna → Colector):

1. Mismo procedimiento

2. Longitud: desde válvula salida hasta recipiente

3. Este tubo puede ser transparente (Tygon)
   para visualizar flujo

CONEXIONES ADICIONALES:

- Tubos para manómetros (cortos, ~10cm)
- Tubo de venteo (hasta recipiente seguro)
- Bypass (si aplica)
```

### Paso 5.4: Fijación de Tubería

```
1. Evitar tensión en conexiones:
   - Tubos deben tener curvas suaves
   - Radio mínimo de curvatura: 50mm
   - Sin torceduras

2. Usar abrazaderas plásticas:
   - Fijar tubos a estructura de soporte
   - Cada 30-40cm
   - No apretar excesivamente

3. Identificar tubos:
   - Etiquetas: "ENTRADA", "SALIDA", "VENTEO"
   - Código de colores (opcional)
```

## FASE 6: Pruebas de Funcionamiento

### Paso 6.1: Prueba de Estanqueidad en Vacío

```
OBJETIVO: Detectar fugas de aire (más fácil que detectar agua)

1. Cerrar válvula de salida (Posición OFF)

2. Cerrar válvula de venteo

3. Conectar bomba de vacío en puerto de entrada:
   - O usar jeringa grande (50ml)

4. Aplicar vacío moderado (-0.3 bar)

5. Inspección:
   - Escuchar silbidos (entrada de aire)
   - Aplicar agua jabonosa en:
     * Interfaz tapa-cuerpo
     * Conexiones roscadas
     * Base del permeámetro
   - Observar formación de burbujas

6. Criterio de aceptación:
   ✓ Sin burbujas
   ✓ Vacío se mantiene 5 minutos
   ✗ Si hay fugas: identificar y corregir
```

### Paso 6.2: Prueba Hidrostática

```
OBJETIVO: Verificar sellado bajo presión real

PROCEDIMIENTO:

1. Abrir válvula de venteo

2. Llenar columna con agua limpia:
   - Usar embudo en puerto de entrada
   - Llenar lentamente
   - Permitir salida de aire por venteo

3. Cuando salga agua por venteo:
   - Cerrar válvula de venteo

4. Conectar bomba peristáltica

5. Aumentar presión gradualmente:
   
   Presión     Tiempo      Inspección
   ────────────────────────────────────
   0.5 bar  →  5 min   →  Visual completa
   1.0 bar  →  5 min   →  Visual + tactil
   1.5 bar  →  10 min  →  Visual detallada
   2.0 bar  →  10 min  →  Verificación final

6. En cada nivel de presión:
   □ Inspeccionar interfaz tapa-cuerpo
   □ Verificar manómetros (lecturas estables)
   □ Tocar conexiones (detectar humedad)
   □ Observar nivel de agua (debe ser constante)

7. Criterio de aceptación:
   ✓ CERO fugas visibles
   ✓ Presión constante durante 10 min
   ✓ Sin gotas en ninguna parte
   ✓ Lectura de manómetros estable
```

### Paso 6.3: Prueba de Flujo

```
1. Abrir válvula de salida (ON)

2. Ajustar flujo con válvula de aguja:
   - Comenzar cerrada
   - Abrir gradualmente
   - Observar lectura de manómetros

3. Medir caudal:
   - Recolectar efluente en probeta graduada
   - Cronometrar (ej. 1 minuto)
   - Calcular: Q (ml/min)

4. Verificar operación de bypass (si existe):
   - Cambiar posición de válvula 3 vías
   - Verificar redirección de flujo

5. Verificar sistema de venteo:
   - Cerrar entrada
   - Abrir venteo
   - Debe aliviar presión rápidamente
```

### Paso 6.4: Prueba con Muestra de Suelo (Opcional)

```
1. Preparar muestra de suelo:
   - Suelo homogéneo, tamizado
   - Humedad conocida
   - Compactación definida

2. Llenar columna:
   - Vaciar agua
   - Agregar suelo en capas (50mm por capa)
   - Compactar uniformemente
   - Hasta altura deseada (200-250mm)

3. Saturar suelo:
   - Llenar desde abajo (evita aire atrapado)
   - Presión baja (0.1-0.2 bar)
   - Hasta que salga agua clara por arriba

4. Realizar ensayo de permeabilidad:
   - Aplicar carga hidráulica constante
   - Medir caudal de salida
   - Calcular coeficiente de permeabilidad

5. Monitorear sellado:
   - Durante todo el ensayo (1-4 horas)
   - Inspeccionar interfaz cada 30 min
   - Documentar cualquier fuga
```

## FASE 7: Ajustes Finales

### Paso 7.1: Corrección de Fugas Menores

```
SI SE DETECTAN FUGAS:

Fuga en tapa-cuerpo:
1. Drenar sistema
2. Aflojar pernos
3. Verificar O-ring:
   - ¿Está torcido?
   - ¿Tiene corte?
   - ¿Está correctamente lubricado?
4. Reposicionar o reemplazar O-ring
5. Reapretar siguiendo secuencia

Fuga en conexión roscada:
1. Aflojar conexión
2. Agregar más cinta PTFE (1-2 vueltas)
3. Reapretar con torque apropiado
4. Si persiste: reemplazar O-ring o empaque

Fuga en válvulas:
1. Verificar cierre completo
2. Si gotea cerrada: reemplazar empaque
3. Consultar manual de válvula
```

### Paso 7.2: Nivelación Final

```
1. Colocar permeámetro en posición de trabajo

2. Usar nivel de burbuja:
   - Verificar en dos direcciones perpendiculares
   - Debe estar perfectamente vertical

3. Si tiene base con patas ajustables:
   - Ajustar tornillos niveladores
   - Bloquear con contratuercas

4. Importancia:
   - Flujo uniforme en el suelo
   - Evita canalización
   - Lecturas precisas
```

### Paso 7.3: Etiquetado y Documentación

```
1. Etiquetar componentes:
   - Válvulas: "ENTRADA", "SALIDA", "VENTEO"
   - Manómetros: "P1 ENTRADA", "P2 SALIDA"
   - Indicar dirección de flujo con flechas

2. Documentar configuración:
   - Fotografiar sistema completo
   - Anotar configuración de válvulas
   - Registrar lecturas de presión de prueba

3. Crear hoja de datos del equipo:
   - Fecha de ensamblaje
   - Componentes utilizados
   - Resultados de pruebas
   - Responsable de ensamblaje
```

## Checklist Final de Ensamblaje

```
ESTRUCTURA
□ Tapa firmemente sujeta al cuerpo
□ Pernos apretados a 5 Nm
□ Sin separación visible en interfaz
□ Sistema de brida intacto

SELLADO
□ O-ring correctamente instalado
□ Empaque secundario en posición
□ Cero fugas en prueba hidrostática
□ Sellado verificado a 2 bar

PUERTOS Y CONEXIONES
□ Insertos roscados firmes
□ Difusor correctamente montado
□ Rejilla filtrante instalada
□ Todas las conexiones sin fugas

VÁLVULAS E INSTRUMENTACIÓN
□ Válvulas operan suavemente
□ Manómetros leen correctamente (0 bar sin presión)
□ Sistema de venteo funcional
□ Todas las válvulas identificadas

TUBERÍA
□ Tubos correctamente conectados
□ Sin torceduras ni tensión
□ Abrazaderas instaladas
□ Dirección de flujo clara

FUNCIONALIDAD
□ Prueba de vacío exitosa
□ Prueba hidrostática a 2 bar exitosa
□ Flujo continuo sin obstrucciones
□ Permeámetro nivelado

DOCUMENTACIÓN
□ Componentes etiquetados
□ Sistema fotografiado
□ Hoja de datos completada
□ Manual de operación disponible
```

FASE 4: Ensamblaje tapa-cuerpo    1 hora

**Con experiencia:** 6-8 horas

## Solución de Problemas Durante Ensamblaje

| Problema | Solución |
|----------|----------|
| Pernos no atornillan | Verificar alineación de perforaciones |
| Tapa no asienta completamente | Verificar que O-ring no esté fuera del canal |
| O-ring se sale al apretar | Aplicar más lubricante, reducir torque |
| Fuga persistente en interfaz | Verificar planicidad de superficies |
| Inserto se afloja | Reinstalar con método térmico |
| Rosca en plástico se daña | Usar inserto roscado, no forzar |
| Válvula gotea | Reemplazar empaque de vástago |
| Manómetro no lee 0 | Presionar botón ZERO o TARE |
